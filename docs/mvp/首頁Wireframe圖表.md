## 互動流程圖

```
用戶進入首頁
    ↓
查看自選股健康度評分
    ↓
瀏覽國際市場動態
    ↓
關注熱門股排行
    ↓
選擇互動方式：
    ├─ 點擊自選股分析 → 進入詳細分析頁面
    ├─ 點擊國際新聞 → 進入新聞詳情頁面
    ├─ 點擊熱門股票 → 進入股票分析頁面
    ├─ 點擊快速問題 → 發送到聊天介面
    └─ 輸入自定義問題 → 發送到聊天介面
```

## 設計要點

### 1. 問題導向設計
- 每個區塊都用問題包裝
- 自然引導用戶點擊互動
- 降低使用門檻

### 2. 視覺層次
- 重要資訊優先顯示
- 使用顏色和圖標區分
- 清晰的視覺層次

### 3. 響應式設計
- 手機優先設計
- 支援平板和桌面
- 保持一致的用戶體驗

### 4. 互動設計
- 明確的點擊目標
- 即時反饋
- 流暢的轉場動畫
